# 数据目录

本目录用于存放医学影像数据集。

## 支持的数据集

### 1. ChestMNIST (胸部X光)
- **描述**: 胸部X光多标签分类数据集，包含14种疾病
- **来源**: NIH ChestX-ray14 数据集的子集
- **任务**: 多标签分类
- **类别数**: 14
- **图像尺寸**: 28x28 / 64x64 / 128x128 / 224x224
- **样本数**: 训练集 73,126 / 验证集 10,000 / 测试集 25,536

### 2. DermaMNIST (皮肤镜图像)
- **描述**: 皮肤病变分类数据集
- **来源**: HAM10000 数据集
- **任务**: 多分类
- **类别数**: 7
- **图像尺寸**: 28x28 / 64x64 / 128x128 / 224x224
- **样本数**: 训练集 7,007 / 验证集 1,003 / 测试集 2,005

### 3. PneumoniaMNIST (肺炎检测)
- **描述**: 儿童胸部X光肺炎检测
- **来源**: 广州市妇女儿童医疗中心
- **任务**: 二分类
- **类别数**: 2 (正常 / 肺炎)
- **图像尺寸**: 28x28 / 64x64 / 128x128 / 224x224
- **样本数**: 训练集 4,708 / 验证集 524 / 测试集 624

## 数据下载

数据集会在首次运行时自动下载到本目录。

也可以手动下载：
```bash
# 安装 medmnist 包
pip install medmnist

# Python 中使用
import medmnist
# 数据集会自动下载到 ~/.medmnist/ 或指定目录
```
